**Note:** Assuming you have downloaded [an installed build][1] (aka pre-compiled binaries) of UE5 for linux

<H2 TopBRCount={1}>Editor Setup</H2>

1. Go to `Edit -> Editor Preferences -> Application Scale` and set it to 0.85 or your preferred value  

1. Go to `Edit -> Editor Preferences -> Enable AutoSave` and uncheck it

1. Go to `Edit -> Editor Preferences -> Enable Pin Value Inspection Tooltips` and uncheck it (If Enabled it causes glitching while handling blueprint nodes)


<H2 TopBRCount={1}>C++ Setup</H2>
1. Set up absolute #include paths in `*.build.cs` file as <a href={UE_ABSOLUTE_INCLUDE_PATH_URL}>instructed here</a>

1. Add absolute include path in `.vscode/c_cpp_properties.json` for vscode:
   ```json
    "includePath": [
        "${default}",
        "/Path/To/Project/Source/ProjectName"
    ]
   ```
