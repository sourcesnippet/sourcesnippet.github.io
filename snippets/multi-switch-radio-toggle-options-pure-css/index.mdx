```CSS
.wrapper {
    background-color: black;
    display: inline-block;
    padding: 1rem;
}

input[type="radio"] {
    display: none;
}

input[type="radio"]:checked+img {
    background-color: rgba(255, 255, 255, 0.3);
}

img {
    width: 3rem;
    height: 3rem;

    padding: 0.75rem;
    margin-right: 1rem;
    border-radius: 50%;

    cursor: pointer;
    object-fit: contain;
    overflow: visible;

    transition: background-color 0.3s ease-out;
}

img:hover {
    background-color: rgba(255, 255, 255, 0.2);
}
```
