Draw Debug shapes in Unreal Engine 5 C++

Written on: 19 June 2025

Point

DrawDebugPoint(
    GetWorld(),                   // World
    FVector(0.0f, 0.0f, 100.0f),  // Location
    15.0f,                        // Point size
    FColor::Red,                  // Color
    false,                        // Persistent?             Default = false
    10.0f,                        // Life Time (Seconds)     Default = -1.f
    0                             // Depth Priority          Default = 0
);
