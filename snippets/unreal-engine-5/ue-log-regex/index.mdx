## To Remove All Prefix
```regex
\[[^\]]*\]\[[^\]]*\]LogTemp: Warning: 
```

#### Example
```plaintext
[2025.05.30-03.31.14:816][520]LogTemp: Warning: Some Message 1
[2025.05.30-03.31.14:816][520]LogTemp: Warning: Some Message 2
[2025.05.30-03.31.14:816][520]LogTemp: Warning: Some Message 3
[2025.05.30-03.31.14:816][520]LogTemp: Warning: Some Message 4
[2025.05.30-03.31.14:816][520]LogTemp: Warning: Some Message 5


Some Message 1

Some Message 5
```

## To Remove date and time

```regex
^\[[^\]]+\]
```

#### Example
