Setter Getter functions in Unreal Engine C++


This will expose the variable Value to Blueprints despite being private and each time it is modified in Blueprints it will invoke the setter getter functions

public:
    UFUNCTION(BlueprintCallable) // Setter must be BlueprintCallable
    void SetValue(int NewValue);

    UFUNCTION(BlueprintPure) // Getter must be BlueprintPure
    int GetValue();

private:
    UPROPERTY(BlueprintSetter=SetValue, BlueprintGetter=GetValue)
    int Value = 10;
